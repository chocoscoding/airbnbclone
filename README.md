## AIRBNB CLONE ##

This is a mini [Airbnb clone](https://airbnbclone-nu.vercel.app/), with full functionalities like:

    -- Listing properties
    -- Managing properties
    -- Creating Reservations
    -- Liking Properties
    
and so much more.

This project was build with Next Js, Tailwind css, Typescript, Prisma and Mongo Db
